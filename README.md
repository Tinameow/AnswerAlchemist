# 510Project
UIUC CS510 Project

### Backend

Open a terminal

```shell
cd backend
pip install -r requirements.txt
python3 server.py
```

The backend will run on http://127.0.0.1:5000

### Frontend

Open another terminal 

```shell
cd frontend
npm install
npm start
```

The backend will run on  http://127.0.0.1:3000
